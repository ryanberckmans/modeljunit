The Data Structures Library in Java

JDSL Tutorial

The JDSL Tutorial consists of a collection of sample programs that demonstrate major concepts and programming styles of the JDSL. Each tutorial consists of a working sample program and a line-by-line explanation. Each lesson introduces, by example, a new concept of the JDSL. Please look in the JDSL javadocs for detailed documentation.

  • Lesson 1: Echo - a simple program that repeats words.
  • Lesson 2: Sequence Fun - alphabetize and scramble a list of words.
  • Lesson 3: Auction - an application thst models bids at an auction.
  • Lesson 4: Triage - models prioritizing medical patients.
  • Lesson 5: Phone Book - a class that models bids at an auction.
  • Lesson 6: Tree Drawing - an application that draws a tree.
  • Lesson 7: Flights - finds shortest time flight itineraries between cities.
The tutorial was written by Robert Cohen and Lucy Perry.
